Hybrid Password Hardening Encryption (HPHE) — Python Implementation

This project provides a practical and fully functional implementation of the Hybrid Password Hardening Encryption (HPHE) protocol.

Localtest:
Python Version
Python 3.8 or higher

PyCryptodome (for AES-256-GCM)
pycryptodome == 3.20.0

Install PyCryptodome:
pip install pycryptodome==3.20.0


Hybrid PHE (HPHE) – WebSocket Demo Implementation

This repository provides a fully functional WebSocket-based implementation of the Hybrid Password-Hardened Encryption (Hybrid PHE / HPHE) protocol.

Python Version

Python 3.10+ recommended
(Minimum: Python 3.8 because pow(a, -1, mod) is used)

Install dependencies:
pip install websockets==12.0
